# 🎯 Breakpoint Quick Commands

## START DEBUGGING

**Terminal 1: Start the backend**
```bash
cd /Users/viyangchaudhari/Projects/youtube
python3 -m backend.src.main
```

**Terminal 2: Trigger a request (hits breakpoint)**
```bash
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}'
```

The program will PAUSE in Terminal 1 showing `(Pdb)` prompt.

---

## PDB COMMANDS AT BREAKPOINT

### Step Through Code
```
n       Next line
s       Step into function
r       Return from function
c       Continue to next breakpoint
```

### Inspect Variables
```
p var           Print variable
pp var          Pretty print
p dir(obj)      List object attributes
p locals()      All local variables
```

### View Code
```
l               List current code
l 100,120       List lines 100-120
w               Show where you are
```

### Other
```
h               Help
q               Quit
!python code    Execute Python
```
